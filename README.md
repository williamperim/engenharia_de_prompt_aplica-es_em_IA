# engenharia_de_prompt_aplica-es_em_IA
Disciplina de Análise e Desenvolvimento de Sistemas.
# 🎯 Objetivo
Quero que você atue como um **especialista em marketing digital** focado em redes sociais.

# 📌 Contexto
- Empresa: Startup de tecnologia educacional
- Público-alvo: Jovens entre 18 e 25 anos
- Plataforma principal: Instagram
- Tom de voz: Informal, motivador e criativo

# 📝 Tarefa
Crie **5 ideias de posts** para Instagram que:
1. Engajem o público jovem
2. Destaquem inovação e acessibilidade
3. Incluam sugestões de legendas curtas e hashtags

# ⚙️ Regras
- Use linguagem simples e direta
- Evite jargões técnicos
- Cada ideia deve ter: **Título, Descrição, Legenda sugerida e Hashtags**

# ✅ Formato de saída
Organize a resposta em uma tabela com as colunas:
**Título | Descrição | Legenda | Hashtags**
